About Me

Experience

Jev Suchoi has extensive experience as among others a Solution Architect, Cloud Consultant, Security Consultant, Tech Lead, Infrastructure Engineer, DevOps Engineer and Developer. In his work for different organizations he has accumulated extensive knowledge of Azure, Azure Active Directory and Azure DevOps. With a developer background Jev is able to fully incorporate development principles when designing and implementing cloud solutions. As an automation & security specialist he is a firm believer of the following concepts & principles;

  • Lifelong learning
  • Everything as Code
  • Automate the boring
  • Pets vs Cattle
  • Idempotency
  • Assume breach
  • Cloud-native security
  • Least Privilege

Employment

Currently Jev Suchoi is employed Rubicon: Cloud Advisor as a Cloud Consultant. By performing this role Jev is responsible for the technology aspects of the Azure team of the Cloud Solutions cluster. These technology aspects include but are not limited to;

  • Facilitating & organizing knowledge sharing in and outside of the team
  • Participating on the technology board
  • Participating in proposals for customers with regards to Azure, Azure DevOps & Office 365
  • Translating Enterprise level designs into technical designs, implementation approaches & other tactical documents
  • Participating in implementation of the designed solutions
  • Conducting recruitment interviews with candidates

Because of Jev’s passion for sharing knowledge Jev’s colleagues consider him as the go-to guy for Azure, scripting & Azure DevOps related questions. This blog is the next step in extending the sharing of knowledge to the outside of the organization.
